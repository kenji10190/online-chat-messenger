# UDPチャットメッセンジャー

UDPソケット通信を使用したリアルタイムチャットアプリ(CLI)です。  
複数クライアントとのメッセージのやり取りができ、NTPサーバーと時刻同期をしているため、  
正確なタイプスタンプ付きメッセージ送信が可能です。  

## 主な機能
- リアルタイムでのメッセージのやり取り
- 複数クライアントへメッセージのブロードキャスト配信
- NTPサーバーとの時刻同期
- ユーザー管理
- 非アクティブユーザーの削除(10分間隔)
- マルチスレッド処理による送受信の並行実行
- ユーザー登録時の簡易バリデーション

## 注意事項
- UDPを使用しているため、メッセージ到達保障はありません。
- 同一ネットワーク内での使用を想定しています。
- ファイアウォールでUDPポートとして9001を許可してください。

## 使用方法
準備としてクライアントPCとサーバーPCを用意してください。  
(クライアントとサーバーを同一PCで行うことも可能です)  
また、クライアント側はサーバーPCのIPアドレスを把握する必要があります。  

1. サーバーPCでserver.pyを実行します。
2. クライアントPCでclient.pyを実行します。
3. クライアントPCでサーバーPCのIPアドレスを入力し、ユーザー名を入力します。
4. クライアント間でメッセージのやり取りを行います。
5. ユーザーがアプリを終了したい場合には、「終了」と入力します。
6. サーバーはCtrl + Cでサーバーを終了させることができます。
